# Workout Plan App

A full-stack workout plan application built with React, Node.js, and MongoDB. This app allows users to plan and manage workout plans.

## Features

- Manage workout plans
- Add new workout plans

## Tech Stack

- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB
